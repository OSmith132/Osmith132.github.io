Dissertation Project

My third year dissertation project with the goal to develop a semantic segmentation model using U-Net architecture, which identifies and categorises teeth, fillings, implants, and other dental structures from 2D dental images.

This project addresses the need for automated dental analysis, offering a reliable tool for dental professionals to aid in diagnosis and treatment.

The system will be trained on labeled dental images and validated through metrics like pixel accuracy and IoU score to ensure strong performance and reliability in clinical applications.


This project will be updated when further progress is made.
